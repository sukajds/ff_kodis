# GDS for Kodi

FlaskFarm에서 GDS의 파일 목록을 Kodi로 전송하고, 재생 요청 시 FF 서버의 `ffmpeg`로 트랜스코딩하여 전달하기 위한 플러그인입니다.

## 기능

* Kodi 설정에서 해상도 조정 가능
* 메타데이터 플러그인과 연동하여 포스터 및 썸네일 표시 가능
* srt 자막이 있을 경우, kodi에서 자막 인식
* 이어보기 및 최신 재생 항목 기능
* 구드공 툴 플러그인과 연동하여 방송중/최신영화는 최신순으로 정렬

## Kodi 애드온 설정

* Kodi 애드온에서는 기본적으로 ff주소, API key를 받습니다.
* 추가로 보안을 원하신다면 FF에서 접속 비밀번호를 개인적으로 설정하실 수 있습니다. 빈 칸의 경우 적용되지 않습니다.
* 방송중/최신영화 정렬순을 최신순/파일순 중 선택하실 수 있습니다.
* 최신순은 아래 경로에만 적용됩니다.

  * `VIDEO/방송중/<장르>` 내부 시리즈 폴더
  * `VIDEO/방송중(기타)/<분류>` 내부 폴더
  * `VIDEO/영화/최신/<하위 폴더>` 내부 폴더
  * 위 폴더들 내부의 에피소드/영상 파일
* 메타데이터 표시를 ON하면 썸네일과 에피소드 목록이 표시되지만, off하면 단순하게 파일명만 표시됩니다
* 메타데이터는 자동으로 입혀집니다만, 조회때 마다 검색해서 저장되는지라 많이 느립니다
* 보았음 표시는 기본 95입니다. 적당하게 입력해주세요.
* 해상도를 원본으로 선택시 Direct Play, 그 외의 해상도를 적용시에는 ffmpeg가 트랜스코딩하여 전송합니다.
* 원본보다 높은 해상도 선택시에는 Direct Play가 적용됩니다.

## 인증

* 추가로 보안을 원하신다면 접속 비호를 개인적으로 설정하실 수 있습니다. 빈 칸의 경우 적용되지 않습니다.
* 도커 혹은 네이티브 기준으로 GDS 경로를 설정해주세요. 해당 경로가 애드온의 루트가 됩니다.

## 재생

* ffmpeg경로를 입력하고, 코덱을 설정 후, 인코더 테스트를 하면 현재 시스템에 가장 적당한 인코더를 선택합니다.
* ffmpeg가 설치되어있지 않으실시에는 터미널에서 `apt update && apt -y install sudo && sudo apt -y install ffmpeg`
* 리스트에 AV표시 여부는 Kodi가 아닌 FF에서 결정합니다.

## DB툴

* PLEX의 DB를 kodi용에 맞춰서 재조정.
* PLEX DB경로를 입력하신 후 설정저장, 이후 가져오기 하시면 됩니다.
* 메타데이터/썸네일 안쓰시는 분께는 필요 없는 기능입니다.
